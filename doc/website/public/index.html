The Carneades Argumentation System

  2014-09-03

The Carneades argumentation system provides web-based, collaborative software tools for:

  • (re)constructing the arguments of a debate in an argument graph
  • generating arguments automatically from OWL ontologies, rules and argumentation schemes
  • critically evaluating arguments, using a formal model of structured argument based on Dung abstract argumentation frameworks.
  • visualizing, browsing, navigating and editing argument graphs
  • conducting structured opinion polls, with questions generated automatically by traversing an argument graph
  • evaluating the effects of alternative policies, rules or regulations, using models of legal rules to construct arguments

The Carneades Web App

Carneades is a multi-user Web application with a three-tiered architecture consisting of a relational database backend layer, an application logic layer and a graphical user interface layer. (An earlier version of Carneades, a single-user desktop application called the Carneades Editor, is still available.)

Features:

  • Multi-user Web application, with a three-tier architecture, including a relational database backend.
  • Zero configuration and zero administration, with a built-in database engine and web server.
  • Can be used locally by a single user, without an Internet connection.
  • Automatic argument evaluation, based on ASPIC+ and Dung abstract argumentation frameworks. Cyclic argument graphs are handled.
  • Argument maps are represented using Structured Vector Graphics (SVG)
  • Exporting and interchanging argument graphs, in a variety of formats, including our own Carneades Argument Format (CAF) and the Argument Interchange Format (AIF) of the University of Dundee.
  • A rule language for representing argumentation schemes and theories (“knowledge bases”) to be used for constructing and reconstructing arguments.

Carneades is open source software licensed using the European Union Public License (EUPL v1.1). Note: Subsequent releases of Carneades will use another open source license, the MPL-2.0 license.

Documentation

A draft user manual is available in various formats (PDF, HTML, ePub). For programmers, online documentation of the Carneades APIs is also available.

Getting Started

Prerequisites:

  • Version 7 or better of a Java Runtime Environment (JRE).
  • A recent version of a standards-compliant Web Browser (e.g. Firefox, Chrome, Safari).

To download, install and run the Carneades Web Application

  1. Download the latest release of the Carneades web application, named carneades-impact-2013.zip. Warning: this is not the zip file shown at the top of this page. That zip file contains the source files.
  2. Unzip the carneades-impact-2013.zip file.
  3. Change to the carneades-impact-2013 directory.

  4. Double click on the carneades-webapp.jar file or execute the following command line:\ $ java -jar carneades-webapp.jar

    Warning: Microsoft Windows users may need to take extra steps to configure Java and the Carneades application. See this explanation from Microsoft.

  5. The server will be start and open the home page of the Carneades client in your web browser.

  6. If the home page fails to open, enter this URL into the address bar of your web browser:\ http://localhost:8080/carneades/#/home

  7. A list of projects should be shown. To get started, click on the link for the “Copyright in the Knowledge Economy” project and follow the instructions.

  8. To shutdown the server you may need to end the Java task using your task manager, depending on your operating system. If you started the server from the command line, you can kill the process with Control-C on Unix machines, including Macs. If you started the server on a Mac by double clicking on the jar file, you can shutdown the server by quitting the “routes_selfexe” application in the usual way, with Command-Q.
